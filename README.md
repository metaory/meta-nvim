# Meta Nvim
## A minimal Neovim distribution with no magic

### Requirement
* Neovim (nightly)

### Installation
* Backup your previous configuration
* `git clone https://github.com/metaory/meta-nvim ~/.config/nvim`
* `nvim +PackerSync`
